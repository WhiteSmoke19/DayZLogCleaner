# DayZ Log Cleaner

Небольшая Windows-утилита для очистки логов и crash-файлов DayZ из папки `%LOCALAPPDATA%\DayZ`.

Программа помогает быстро освободить место на диске, когда DayZ после запусков или вылетов оставляет много файлов `.RPT`, crash-логов и дампов памяти.

## Возможности

- Автоматически находит папку DayZ через системный путь Windows `%LOCALAPPDATA%\DayZ`.
- Не привязана к имени пользователя и диску `C:`.
- Работает на разных ПК без Python.
- Финальный `.exe` можно собрать автономным: пользователю не нужно устанавливать .NET Runtime.
- Показывает найденные файлы, их размер и дату изменения.
- Перед удалением запрашивает подтверждение.
- Позволяет вручную выбрать другую папку, если DayZ хранит логи нестандартно.
- Встроенная иконка и простой графический интерфейс.

## Что удаляется

По умолчанию программа удаляет только типичные временные файлы DayZ:

- `*.RPT`
- `*.rpt`
- `*.mdmp`
- `*.bidmp`
- `*.dmp`
- `crash_*.log`
- `CrashReporter.log`

Также есть опция в интерфейсе для дополнительного удаления всех `*.log` файлов в выбранной папке.

## Скачать

Готовый файл для пользователей нужно брать из раздела [Releases](../../releases).

Для обычного использования достаточно скачать `DayZLogCleaner.exe` и запустить его двойным кликом.

## Использование

1. Запустите `DayZLogCleaner.exe`.
2. Проверьте путь к папке DayZ.
3. Нажмите `Сканировать`, если нужно обновить список.
4. Посмотрите, какие файлы будут удалены.
5. Нажмите `Очистить`.
6. Подтвердите удаление.

Если папка DayZ находится в другом месте, нажмите `Выбрать` и укажите нужную папку вручную.

## Системные требования

- Windows x64.
- Для готового релизного `.exe` не требуется Python.
- Для готового релизного `.exe` не требуется установленный .NET Runtime, если файл собран с `--self-contained true`.

## Безопасность

Программа не удаляет всю папку DayZ. Она работает только с заданными шаблонами файлов логов и дампов.

Перед удалением отображается список найденных файлов и общий размер. Удаление выполняется только после подтверждения пользователя.

## Назначение

Утилита создана для ручной очистки локальных логов DayZ. Она не меняет файлы игры, настройки, моды, сохранения или данные Steam.
